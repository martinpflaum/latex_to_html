1 Foundations

Set theory

\addcontentslinetocsubsectionSet theory

1.1 Naive Definition of Sets (Georg Cantor 1882) A set is a gathering together into a whole of definite, distinct objects of our perception or of our thought which are called elements of the set.

1.2 In modern Mathematics, set theory is developed axiomatically. Axiomatic set theory goes back to Zermelo-Fraenkel and forms the basis of modern mathematics, together with formal logic. We follow the axiomatic approach here too, even though we will not introduce all of the set theory axioms of Zermelo-Fraenkel and will not go into the subtleties of set theory.

1.3 Axiomatic Definition of Sets (cf.~Zermelo-Fraenkel 1908, 1973) Sets are denoted by letters of the roman and greek alphabets : a,b,c,\ldots,x,y,z, A,B,C,\ldots,X,Y,Z, \alpha,\beta,\gamma,\delta,\ldots, \chi,\psi, \omega, and A,B,\Gamma, \Delta, \ldots,X,\Psi,\Omega. The fundamental relational symbols of set theory are the equality sign = and the element sign \in. Further symbols of set theory are the numerical constants 0,1,2,\ldots , 9 and the symbol for the empty set \emptyset.

  1. ( Extensionality Axiom) Two sets M and N are equal if and only if they have the same elements. Formally: \forall M \, \forall N \: \big( (M = N) \Leftrightarrow (\forall x \: (x\in M) \Leftrightarrow (x\in N))\big) .
  2. (Definition of the subset relation) A set M is called a subset of a set N, in signs M\subset N, if every element of M is an element of N. Formally: (M\subset N) \Leftrightarrow ( \forall x \: (x\in M) \Rightarrow (x\in N)) .
  3. The extensionality axiom can now be expressed equivalently as follows: \mbox \hspace17mm\forall M \, \forall N \: \big( (M = N) \Leftrightarrow ( ( M \subset N) \: \& \: (N \subset M)) \big) .
  4. ( Axiom of Empty Set) There exists a set which has no elements. Formally: \exists E \, \forall x \: \neg (x \in E) . [1mm] The set having no elements is uniquely determined by the extensionality axiom. It is denoted by \emptyset.
  5. ( Separation Scheme) Let M be a set and P(x) a formula (or in other words a property). Then there exists a set N whose elements consist of all x\in M such that P(x) holds true. In signs: N = \{ x \in M \mid P(x) \} . [1mm] Using the separation scheme one can define the intersection of two sets M and N as M \cap N = \{ x\in M\mid x\in N\} and the complement M \setminus N as the set \{ x\in M \mid x \notin N\}.
  6. ( Axiom of Pairings) For all sets x and y there exists a set containing exactly x and y. Formally: \forall x \, \forall y \, \exists M \, \forall z \big( z \in M \Leftrightarrow (z = x \vee z = y )\big) . [1mm] The set containing x and y as elements (and no others) is denoted \{ x , y \}. If x = y, one writes \{ x \} for this set.
  7. ( Axiom of the Union) Given two sets M and N there exists a set consisting of all elements of M and N (and no others). Formally: \forall M \, \forall N \, \exists U \, \forall x \: \big((x \in U ) \Leftrightarrow ( x \in M \vee x \in N)\big) . [1mm] The set U in this formula is uniquely defined by the extensionality axiom and is called the union of M and N. It is denoted M \cup N. More generally, if M is a set, then there exists a set denoted by \bigcup M consisting of all elements of elements of M. Formally: \forall M \, \exists U \, \forall x \: \big((x \in U ) \Leftrightarrow ( \exists X \: ( X \in M \: \& \: x \in X ))\big) . [1mm] The set U in this formula then is uniquely determined and abbreviated \bigcup M.
  8. ( Power Set Axiom) For each set M there exists a set \mathscr{P}(M) containing all subsets of M (and no others). Formally: \forall M \, \exists P \, \forall x \: \big( (x \subset M) \Leftrightarrow ( x\in P) \big). [1mm] The set P in this formula is uniquely defined by the extensionality axiom and is called the power set of M. It is denoted \mathscr{P}(M).
  9. ( Axiom of Infinity) There exists a set which contains \emptyset and with each element x also the union x \cup \{ x\}. Formally: \exists M \: \big( \emptyset \in M \: \& \: \forall x ( x \in M \Rightarrow x \cup \{x\} \in M) \big) .
  10. ( Axiom of Choice) For any set M of nonempty sets, there exists a choice function f defined on M that is a map M \to \bigcup M such that f(x) \in x for all x \in M. Formally: \forall M \big( \emptyset \notin M \Rightarrow \exists f : M \to \bigcup M, \: \forall x \in X \: ( f(x) \in x) \big) .



Remark 1.4 In the formulation of the Axiom of Choice we used the notion of a function introduced below in \Crefdef:function.



Proposition 1.5 Let L, M, N be sets. Then the following statements hold true.

  1. (commutativity) M \cup N = N \cup M and M \cap N = N \cap M.
  2. ( associativity) M \cup (N \cup L) = (N \cup M) \cup L and M \cap (N \cap L) = (N \cap M) \cap L.
  3. (distributivity) M \cup (N \cap L) = (M \cup N) \cap (M \cup L) and M \cap (N \cup L) = (M \cap N) \cup (M \cap L).
  4. M \cap N \subset M and M\subset M \cup N.
  5. If for a set X the relations X \subset M and X \subset N hold true, then X \subset M \cap N. If for a set Y the relations M \subset Y and N \subset Y are satisfied, then M \cup N \subset Y .
  6. M \setminus \emptyset = M and M \setminus M = \emptyset.
  7. M \setminus (M \cap N) = M \setminus N and M \setminus (M \setminus N) = M \cap N.
  8. (de Morgan's laws) M \setminus (N \cup L) = (M \setminus N) \cap (M \setminus L) and M \setminus (N \cap L) = (M \setminus N) \cup (M \setminus L) .



Cartesian products

\addcontentslinetocsubsectionCartesian products

Definition 1.6 Let X and Y be sets. For all x\in X and y\in Y the pair (x,y) is defined as the set \big\{ \{x \}, \{ x, y \} \big\}. The cartesian product X \times Y is defined as

\big\{ z \in \mathscr{P}(\mathscr{P}(X \cup Y)) \mid \exists x \in X \, \exists y \in Y : \: z = \{ \{x \}, \{ x, y \}\} \big\} .




Proposition 1.7 For sets X,Y and elements x, x'\in X and y,y' \in Y the pairs (x,y) and (x',y') are equal if and only if x = x' and y =y'.



Proposition 1.8 Let L, M, N be sets. Then the following associativity law for the cartesian product is satisfied:

  1. L \times (M \times N) = (L \times M) \times N.

Moreover, the following distributivity laws hold true:

    \setcounterenumi1
  1. L \times (M \cup N) = (L \times M) \cup (L \times N) and (M \cup N) \times L = (M \times L) \cup (N \times L),
  2. L \times (M \cap N) = (L \times M) \cap (L \times N) and (M \cap N) \times L = (M \times L) \cap (N \times L),
  3. L \times (M \setminus N) = (L \times M) \setminus (L \times N) and (M \setminus N) \times L = (M \times L) \setminus (N \times L).



Relations

\addcontentslinetocsubsectionRelations

Definition 1.9 A triple R = (X,Y,\Gamma) with X and Y being sets and \Gamma a subset of the cartesian product X\times Y is called a relation from X to Y. If Y=X, a relation (X,X,\Gamma) is called a relation on X. The set \Gamma is called the graph of the relation. If (x,y) \in \Gamma, on says that x and y are in relation, and denotes that by x \, R \, y.



Remark 1.10 Usually, a relation is often denoted by a symbol like for example \sim, \leq or \equiv. The statement that x and y are in relation is then symbolically written x \sim y, x \leq y, x \equiv y, respectively.



Definition 1.11 A relation \sim on a set X is called an equivalence relation if it has the following properties:

  1. Reflexivity For all x\in X the relation x \sim x holds true.
  2. Symmetry For all x,y \in X, if x \sim y holds true, then y \sim x is true as well.
  3. Transitivity For all x,y,z \in X the relations x\leq y and y \leq x entail x\leq z.





Definition 1.12 A set X together with a relation \leq on it is called an ordered set, a partially ordered set or a poset if the following axioms are satisfied:

  1. Reflexivity For all x\in X the relation x \leq x holds true.
  2. Antisymmetry If x \leq y and y \leq x for some x,y\in X, then x =y.
  3. Transitivity For all x,y,z \in X the relations x\leq y and y \leq x entail x\leq z.

The relation \leq is then called an order relation or an order on X. If in addition Axiom ref_error below holds true, (X,\leq) is called a totally ordered set and \leq a total order on X.

    \setcounterenumi3
  1. Totality For all x,y\in X the relation x \leq y or the relation y \leq x holds true.

A total order relation \leq on X satisfies the following law of trichotomy, where x < y stands for x \leq y and x \neq y:

    \setcounterenumi4
  1. \hspace-10mm( Law of Trichotomy) For all x,y\in X exactly one of the statements x < y, x=y or y < x holds true.





Definition 1.13 Let (X,\leq) be an order set and A\subset X a subset. Then one calls

  1. M\in X a maximum if for all x \in X satisfying M \leq x the relation x=M holds true,
  2. m\in X a minimum if for all x \in X satisfying x\leq m the relation x=m holds true,
  3. G\in X a greatest element if for all x \in X the relation x\leq G holds true,
  4. s\in X a lowest or smallest element if for all x \in X the relation s\leq x holds true,
  5. B\in X an upper bound of A if a \leq B for all a \in A,
  6. b\in X a lower bound of A if b \leq a for all a \in A,
  7. S\in X a supremum of A if S is a least upper bound of A, and finally
  8. i\in X an infimum of A if i is a greatest lower bound of A.

If A has an upper bound it is called bounded above, if it has alower boudn one says it is bounded below. A subset A \subset X bounded above and below is called a bounded subset of X.



Remarks 1.14

  1. A greatest or a lowest element is always uniquely determined, but such elements might not exist or just one of them. Likewise, the supremum and the infimum of a subset A \subset X are uniquely determined, but possibly do not exist. If existent, they are denoted by \sup A and \inf A, respectively.
  2. A greatest element is always maximal, but in general not vice versa. The same holds for minimal and least elements that is a least element is always minimal but a minimal element is in general not a least element.



Functions

\addcontentslinetocsubsectionFunctions

Definition 1.15 By a function f one understands a triple (X,Y,\Gamma) consisting of

  1. a set X, called the domain of the function,
  2. a set Y, called the range or target of the function,
  3. a set \Gamma of pairs (x,y) of points x\in X and y\in Y, called the graph of the function, such that for each x\in X there is a unique f(x)\in Y with \big( x,f(x)\big) \in \Gamma.

A function f with domain X, range Y and graph \Gamma = \{ (x,y) \in X \times Y \mid y = f(x) \} will be denoted

f:X \rightarrow Y, \: x \mapsto f(x) .




Example 1.16 The following are examples of functions:

  1. the identity function on a set X, \operatorname{id}_X : X \rightarrow X, x\mapsto x,
  2. polynomial functions which are functions of the form p: \mathbb{R} \to \mathbb{R}, x \mapsto \sum_{k=0}^n a_k x^k, where the a_k, k= 0, \ldots ,n are real numbers called the coefficients of the polynomial function,
  3. the absolute value function | \cdot | :\mathbb{R} \rightarrow \mathbb{R}, x \mapsto |x| = \sqrt{x^2}, [2mm] the euclidean norm \| \cdot \| :\mathbb{R}^2 \rightarrow \mathbb{R}, (x,y) \mapsto \sqrt{x^2 + y^2} on \mathbb{R}^2 and more generally the euclidean norm \| \cdot \| :\mathbb{R}^n \rightarrow \mathbb{R}, (x_1,\ldots , x_n) \mapsto \sqrt{\sum_{i=10}^nx_i^2} on \mathbb{R}^n.

Further examples of functions defined on (subsets of) \mathbb{R} are the exponential funtion, the logarithm, the trigonometric functions, and so on. Precise definitions of these will be introduced later.



Definition 1.17 Functions of the form f:\:\mathbb{R}^m \rightarrow \mathbb{R}^n such that

  1. f (0) = 0 and
  2. f(v+w) = f(v) + f(w) for all v,w \in \mathbb{R}^m

are called linear. A function g: \:\mathbb{R}^m \rightarrow \mathbb{R}^n is called affine if there exists an a \in \mathbb{R}^n and a linear function f: \:\mathbb{R}^m \rightarrow \mathbb{R}^n such that g(v) = f(v)+ a for all v\in \mathbb{R}^m.



Definition 1.18 A function f: X \to Y is called

  1. injective or one-to-one if for all x_1,x_2 \in X the equality f(x_1)= f(x_2) implies x_1 = x_2,
  2. surjective or onto if for all y\in Y there exists an x\in X such that f(x)= y, and
  3. bijective if it is injective and bijective.





Definition 1.19 Let f : X \to Y and g: Y \to Z be two functions. The composition g \circ f : X \to Z then is defined as the function with domain X, range Z and graph \Gamma = \{ (x,z) \in X \times Z \mid z = g(f(x)) \} . In other words, g\circ f maps an element x\in X to g(f(x))\in Z.



Definition 1.20 A function f : X \to Y is called invertible if there exist a function g: Y \to X such that g \circ f = \mathrm{id}_X and f \circ g = \mathrm{id}_Y.



Theorem 1.21 A function f : X \to Y is invertible if and only if it is bijective.



Definition 1.22 Let f: X \to Y be a function. For every subset A \subset X one defines the image of A under f as the set

f(A) = \{ f(x) \in Y\mid x \in A \} .
The set f(X) is called the image of the function f and is usually denoted by the symbol \operatorname{im} (f). In case B is a subset of Y, the preimage of B under f is defined as the set

f^{-1} (B) \subset X .




Remarks 1.23

  1. The notation f^{-1} in the preceding definition does not mean that f is invertible. The preimage map f^{-1} associated to a function f : X \to Y has domain \mathscr{P}(Y) and range \mathscr{P}(X), whereas the inverse map g^{-1} of an invertible function g: X \to Y has domain Y and range X. The context will always make clear which of these two functions is meant when using the ``to the power negative one'' symbol on functions. If f : X \to Y is invertible, the preimage map f^{-1} : \mathscr{P}(Y) \to \mathscr{P}(X) and the inverse map f^{-1} : Y \to X are related by

    f^{-1} (\{y\}) = \big\{ f^{-1} (y) \big\} \quad \text{for all } y \in Y .
  2. The image \operatorname{im} (f) of a function f:X \to Y is always contained in the range Y. Equality \operatorname{im} (f) = Y holds if and only if f is surjective.





Proposition 1.24 For a function f: X \to Y between two sets X and Y the following statements hold true for the images respectively preimages under f of subsets A, A_i, E \subset X with i\in I and B, B_j, F\subset Y with j\in J:

  1. The relations

    f^{-1}\big( f (A)\big) \supset A \quad \text{and} \quad f\big( f^{-1} (B)\big) = B \cap \operatorname{im} (f) \subset B
    hold true.
  2. The image map f : \mathscr{P}(X) \to \mathscr{P}(Y) preserves unions that is

    f \left( \bigcup_{i\in I} A_i \right) = \bigcup_{i\in I} f (A_i) .
  3. In regard to intersections, the image map f : \mathscr{P}(X) \to \mathscr{P}(Y) fulfills the relation

    f \left( \bigcap_{i\in I} A_i \right) \subset \bigcap_{i\in I} f (A_i) .
  4. With respect to set-theoretic complement the image map f : \mathscr{P}(X) \to \mathscr{P}(Y) acts as follows:

    f (A) \setminus f(E) \subset f(A\setminus E) \quad \text{if } E \subset A .
  5. The preimage map f^{-1}: \mathscr{P}(Y) \to \mathscr{P}(X) preserves unions that is

    f^{-1}\left( \bigcup_{j\in J} B_j \right) = \bigcup_{j\in J} f^{-1} (B_j) .
  6. The preimage map f^{-1}: \mathscr{P}(Y) \to \mathscr{P}(X) preserves intersections that is

    f^{-1}\left( \bigcap_{j\in J} B_j \right) = \bigcap_{j\in J} f^{-1} (B_j) .
  7. The preimage map f^{-1}: \mathscr{P}(Y) \to \mathscr{P}(X) acts as follows with respect to set-theoretic complement:

    f^{-1} (B) \setminus f^{-1}(F ) = f^{-1} (B\setminus F) \quad \text{if } F \subset B .





Remark 1.25 The image map f : \mathscr{P}(X) \to \mathscr{P}(Y) does in general not preserve intersections. To see this consider the square function f: \mathbb{R} \to \mathbb{R}, x \mapsto x^2. Then

f \big( (-\infty,0] \cap [0,\infty)\big) = \{ 0\} \quad \text{but} \quad f \big( (-\infty,0] \big) \cap f \big( [0,\infty)\big) = [0,\infty ) ¸.
If f is an injective map, the corresponding preimage map preserves all intersections. Actually this is even a sufficient criterion for injectivity of f.

Basic algebraic structures

\addcontentslinetocsubsectionBasic algebraic structures

Definition 1.26 A set G together with a binary operation * : \: G \times G \to G and a distinguished element e \in G is called a group if the following axioms hold true

  1. (associativity) g * (h *k) = (g*h) *k for all g,h,k \in G.
  2. (neutrality of 0) g* e = e*g = g for all g\in G.
  3. (existence of inverses) For every g\in G there exists an element h\in G such that g* h = h*g = e.

If in addition the following property holds true, the group G is called abelian.

    \setcounterenumi3
  1. (commutativity) g*h = h*g for all g,h \in G.



2 Number Systems

Natural numbers

\addcontentslinetocsubsectionNatural numbers

Definition 2.1[Peano] A triple (\mathbb{N},0,s) consisting of a set \mathbb{N}, an element 0\in \mathbb{N} called zero element and a map s: \mathbb{N} \rightarrow \mathbb{N} called successor map is called a system of natural numbers if the following axioms hold true:

  1. 0 is not in the image of s .
  2. s is injective.
  3. (Induction Axiom) Every inductive subset of \mathbb{N} coincides with \mathbb{N} , where by an inductive subset of \mathbb{N} one understands a set I \subset \mathbb{N} having the following properties:

  4. 0 is an element of I .
  5. If n \in I , then s (n) \in I .

\endenumerate

By Axiom ref_error, 0 is not in the image of the successor map. But all other elements of the Peano structure are.

Proposition 2.2 Let (\mathbb{N},0,s) be a system of natural numbers. Then the image of s coincides with the set \mathbb{N}_{\ne 0}:=\{ n \in \mathbb{N} \mid n \neq 0 \} of all non-zero elements, in signs s (\mathbb{N}) = \mathbb{N}_{\ne 0}.



Theorem 2.3 The set \mathbb{N} of natural numbers together with addition + : \: \mathbb{N} \times \mathbb{N} \to \mathbb{N}, multiplication \cdot : \: \mathbb{N} \times \mathbb{N} \to \mathbb{N} and the elements 0 and 1 := s(0) satisfies the following axioms:

  1. (associativity) l + (m + n) = (l+m) +n for all l,m,n \in \mathbb{N}.
  2. (commutativity) m + n = n+m for all m,n \in \mathbb{N}.
  3. (neutrality of 0) m +0 = 0 +m = m for all m\in \mathbb{N}.

  1. (associativity) l \cdot (m \cdot n) = (l\cdot m) \cdot n for all l,m,n \in \mathbb{N}.
  2. (commutativity) m \cdot n = n\cdot m for all m,n \in \mathbb{N}.
  3. (neutrality of 1) m \cdot 1 = 1 \cdot m = m for all m\in \mathbb{N}.

  1. (distributivity) l \cdot (m +n ) = (l \cdot m) + (l \cdot n) \quad and (m +n ) \cdot l = (m \cdot l) + (n \cdot l) \quad for all l,m,n\in \mathbb{N}.

In other words, \mathbb{N} together with + and \cdot and the elements 0,1 is a semiring.

Real numbers

\addcontentslinetocsubsectionReal numbers

Definition 2.4 By a field of real numbers one understands a set \mathbb{R} together together with binary operations + : \: \mathbb{R} \times \mathbb{R} \to \mathbb{R} and \cdot : \: \mathbb{R} \times \mathbb{R} \to \mathbb{R} called addition and multiplication, two distinct elements 0 and 1 and an order relation \leq such that the following axioms are satisfied:

  1. (associativity) x + (y + z) = (x+y) +z for all x,y,z \in \mathbb{R}.
  2. (commutativity) x + y = y+x for all x,y \in \mathbb{R}. .
  3. (neutrality of 0) x +0 = 0 +x = x for all x\in \mathbb{R}.
  4. (additive inverses) For every x \in \mathbb{R} there exists y \in \mathbb{R}, called negative of x such that x +y = y +x = 0. The negative of x is usually denoted -x.

  1. (associativity) x \cdot (y \cdot z) = (x\cdot y) \cdot z for all x,y,z \in \mathbb{R}.
  2. (commutativity) x \cdot y = y\cdot x for all x,y \in \mathbb{R}.
  3. (neutrality of 1) x \cdot 1 = 1 \cdot x = x for all x\in \mathbb{R}.
  4. (multiplicative inverses of nonzero elements) For every x \in \mathbb{R}^* := \mathbb{R} \setminus \{ 0 \} there exists y \in \mathbb{R}, called inverse of x such that x \cdot y = y \cdot x = 1. The inverse of x \neq 0 is usually denoted x^{-1}.

  1. (distributivity) x \cdot (y +z ) = (x \cdot y) + (x \cdot z) \quad and (x +y ) \cdot z = (x \cdot z) + (y \cdot z) \quad for all x,y,z\in \mathbb{R}.

    \setcounterenumi4
  1. (monotony of addition) For all x,y,z \in \mathbb{R} the relation x \leq y implies x +z \leq y+z.
  2. (monotony of multiplication) For all x,y,z \in \mathbb{R} with z \geq 0 the relation x \leq y implies x \cdot z \leq y \cdot z.

  1. (completeness) Each non-empty subset X\subset \mathbb{R} bounded above has a least upper bound.

In other words, \mathbb{R} together with + and \cdot, the elements 0,1 and the order relation \leq is a complete ordered field.



Theorem 2.5 There exists (up to isomorphism) only one field of real numbers \mathbb{R}.

3 Analysis of Functions of One Real Variable

\subsectionLimits and Continuity

Definition 3.1 Let (x_n)_{n\in \mathbb{N}} be a sequence of real numbers. The sequence is said to converge to a real number x\in \mathbb{R} if for every \varepsilon > 0 there exists an N\in \mathbb{N} such that

| x_n - x | < \varepsilon \: \text{for all natural } n \geq N .
One then calls (x_n)_{n\in \mathbb{N}} a convergent sequence and x its limit.



Proposition 3.2 The limit of a convergent sequence is uniquely determined.



Definition 3.3 The function f: \mathbb{R} \rightarrow \mathbb{R} has the limit b at the point a \in \mathbb{R}, written

\lim_{x \rightarrow a} f(x) = b ,
if for every \varepsilon > 0 there is a \delta > 0 such that

| f (x) - b | < \varepsilon \text{ for all } x \neq a \text{ with } | x - a | < \delta .




Remark 3.4 Intuitively, \lim_{x \rightarrow a} f(x) = b means that f(x) is as close to b as we wish whenever the distance of the point x to a is sufficiently small.



Definition 3.5 A function f: D \rightarrow \mathbb{R} defined on a subset D \subset \mathbb{R} is called continuous at the point a \in \mathbb{R}, if

\lim_{x \rightarrow a} f(x = f(a) .
The function f : D \rightarrow \mathbb{R} is said to be continuous on D or just continuous if it is continuous at every point a \in D.

\subsectionDifferentiability

Definition 3.6 A function f:I \rightarrow \mathbb{R}, x \mapsto f(x) defined on an open interval I\subset \mathbb{R} is called differentiable in a \in I if the limit

f'(x) := \lim_{h\rightarrow 0} \frac{ f (a+h) - f(a)}{h}


exists. One then calls f' (x) the derivative of f at a. The derivative of a function f differentiable at a is sometimes also denoted by Df (a) or \frac{d f}{d x} (a). If f is differentiable in every point of its domain I, then one says that f is differentiable.



Proposition 3.7 Let f,g:I \to \mathbb{R} be two functions defined on the open interval I \subset \mathbb{R}. Ass ume that both f,g are differentiable in the point a\in I. Then the following holds true:

  1. The sum f+g :I \to \mathbb{R}, x \mapsto f(x)+g(x) is differentiable in a with derivate given by (f+g)'(a)=f'(a)+g'(a).
  2. For every c\in \mathbb{R} the scalar multiple cf :I \to \mathbb{R}, x \mapsto c \, f(x) is differentiable in a with derivate given by (cf)'(a)=c\, f'(a).
  3. (Product rule) The product f \cdot g :I \to \mathbb{R}, x \mapsto f(x)\cdot g(x) is differentiable in a with derivative given by (f\cdot g)'(a)=f'(a) \cdot g(a) + f(a) \cdot g'(a).





Proposition 3.8 Let f:I \to \mathbb{R} and g:J \to \mathbb{R} be two functions defined on open intervals I,J \subset \mathbb{R}. Assume that f(I) \subset J. If f is differentiable in some point a \in I and g is differentiable in the pont f(a), then the composition g\circ f : I \to \mathbb{R}, x \mapsto g(f(x)) is differentiable in a with derivative

(g\circ f)'(a) = g'(f(a)) \cdot f'(a) .




Examples 3.9 The following is a list of differentiable functions and their derivatives.

  1. Every polynomial function

    p:\mathbb{R}\to\mathbb{R}, \quad x \mapsto p(x)= \sum_{k=0}^n a_k x^k ,
    where a_0,\ldots, a_n \in \mathbb{R} are its coefficients, is differentiable and has derivative

    p' : \mathbb{R} \to \mathbb{R}, \quad x \mapsto \sum_{k=1}^n k \, a_k x^{k-1} .
    In particular the monomials q_n: \mathbb{R} \to \mathbb{R}, x \mapsto x^n with n\in \mathbb{N} are differentiable with derivatives given by q_n': \mathbb{R} \to\mathbb{R}, x \mapsto n \, x^{n-1}.
  2. The trigonometric functions \sin, \cos, \tan, \cot are all differentiable on their natural domains. The derivatives are given by

    \sin'(x) = \cos x \hspace{8mm} \text{for } x \in \mathbb{R},

    \cos'(x) = -\sin x \hspace{4.7mm} \text{for } x \in \mathbb{R},

    \tan'(x) = \frac{1}{\cos^2 x} \hspace{5.5mm} \text{for } x \in \mathbb{R} \setminus \{ (2k+1)\frac{\pi}{2} \mid k \in \mathbb{N} \},

    \cot'(x) = \frac{-1}{\sin^2 x} \hspace{6mm} \text{for } x \in \mathbb{R} \setminus \{ k \pi \mid k \in \mathbb{N} \}.



\subsectionBasic curve analysis

Definition 3.10 (Symmetries of a function) A function f:\mathbb{R} \to \mathbb{R}, x \mapsto y =f(x) is called even if its graph is symmetric to the y-axis meaning that f(-x)=f(x) for all x\in \mathbb{R}. The function f is called odd if its graph is symmetric to the origin that is if f(-x)=-f(x) for all x\in \mathbb{R}.



Definition 3.11 A function f:D \rightarrow \mathbb{R} defined on a subset D \subset \mathbb{R} is called strictly monotone increasing, if f(x') < f(x) whenever x' < x for two points x,x'\in D. If instead f(x) < f(x') for all points x,x'\in D with x' < x , then f is called strictly monotone decreasing.



Proposition 3.12 Let f:I \to \mathbb{R} be a differentiable function defined on the open interval I.

  1. If f'(x) > 0 for all x \in I, then f is strictly monotone increasing, if f'(x) < 0 for all x\in I, then f is strictly monotone decreasing.
  2. If for some x\in I the derivative of f in x vansihes that is if f'(x) =0, then the grap of f has in x a horizontal tangent.





Definition 3.13 Let f:I \rightarrow \mathbb{R}, x\mapsto f(x) be a function defined on an open interval I\subset \mathbb{R}. A point a\in I is called a relative maximum (respectively a relative minimum) of f, if f(x)\leq f(a) (respectively f(x)\geq f(a)) for all x in an \varepsilon-neighborhood U_\varepsilon (a) \subset I of a. The point a\in \mathbb{R}^n is called a global maximum (resp.~ global minimum) of f over the region R\subset \mathbb{R}^n , if f(x)\leq f(a) (resp.~f(x)\geq f(a)) for all x\in R.



Theorem 3.14 Assume that f:\mathbb{R}^2 \rightarrow \mathbb{R}, (x,y)\mapsto f(x,y) is a twice continuously partially differentiable function. Suppose that (a,b) is a point where \operatorname{grad} f(a,b)=0. Let

D = \dfrac{\partial^2f}{\partial x^2}(a,b)\cdot \dfrac{\partial^2f}{\partial y^2}(a,b) - \left( \dfrac{\partial^2f}{\partial x \partial y} (a,b) \right)^2 .
\beginitemize \item If D > 0 and \dfrac{\partial^2f}{\partial x^2}(a,b) > 0, then f has a local minimum in a. \item If D > 0 and \dfrac{\partial^2f}{\partial x^2}(a,b) < 0, then f has a local maximum in a. \item If D < 0, then f has a saddle point in a. \item If D=0, no conclusion can be made: f can have a local maximum, a local minimum, a saddle point, or none of these in the point (a,b). \enditemize



Definition 3.15 If f:\mathbb{R}^2 \rightarrow \mathbb{R}, (x,y)\mapsto f(x,y) and g :\mathbb{R}^2 \rightarrow \mathbb{R}, (x,y)\mapsto g(x,y) are functions, and c\in \mathbb{R} a number, a point (a,b) \in \mathbb{R}^2 is called a local maximum (resp.~ local minimum) of f under the constraint g(x,y)=c, if f(x,y)\leq f(a,b) (resp.~f(x,y)\geq f(a,b)) for all (x,y)\in \mathbb{R}^2 near (a,b) which satisfy g(x,y)=c. The point (a,b)\in \mathbb{R}^2 is called a global maximum (resp.~ global minimum) of f under the constraint g(x,y)=c, if f(x,y)\leq f(a,b) (resp.~f(x,y)\geq f(a,b)) for all (x,y)\in \mathbb{R}^2 which satisfy g(x,y)=c.



Theorem 3.16 Assume that f:\mathbb{R}^2 \rightarrow \mathbb{R}, x\mapsto f(x) is a smooth function, and g:\mathbb{R}^2 \rightarrow \mathbb{R}, x\mapsto g(x) a smooth constraint. If f has a maximum or minimum at the point (a,b) under the constraint g(x,y)=c, then (a,b) either satisfies the equations

\operatorname{grad} f (a,b) = \lambda \operatorname{grad} g (a,b) \text{ and } g(a,b)=c \text{ for some $\lambda \in \mathbb{R}$},
or \operatorname{grad} g (a,b)=0. The number \lambda is called the Lagrange multiplier.

4 Analysis of Functions of Several Real Variables

\subsectionLimits and Continuity

Definition 4.1 Let (x_n)_{n\in \mathbb{N}} be a sequence of real numbers. The sequence is said to converge to a real number x\in \mathbb{R} if for every \varepsilon > 0 there exists an N\in \mathbb{N} such that

| x_n - x | < \varepsilon \: \text{for all natural } n \geq N .
One then calls (x_n)_{n\in \mathbb{N}} a convergent sequence and x its limit.



Proposition 4.2 The limit of a convergent sequence is uniquely determined.



Definition 4.3 The function f: \mathbb{R}^2 \rightarrow \mathbb{R} has the limit L at the point (a,b) \in \mathbb{R}^2, written

\lim_{(x,y) \rightarrow (a,b)} f(x,y) = L ,
if for every \varepsilon > 0 there is a \delta > 0 such that

| f (x,y) - L | < \varepsilon \text{ for all $(x,y)\neq (a,b)$ with $d\big ( (x,y),(a,b) \big) < \delta$.}




Remark 4.4 Intuitively, \lim_{(x,y) \rightarrow (a,b)} f(x,y) = L means that f(x,y) is as close to L as we wish whenever the distance of the point (x,y) to (a,b) is sufficiently small.



Definition 4.5 The function f: \mathbb{R}^2 \rightarrow \mathbb{R} is called continuous at the point (a,b) \in \mathbb{R}^2, if

\lim_{(x,y) \rightarrow (a,b)} f(x,y) = f(a,b) .
The function f is said to be continuous on a region R\subset \mathbb{R}^2, if it is continuous at every point (a,b) \in R.

\subsectionDifferentiability in one real dimension

Definition 4.6 A function f:\mathbb{R} \rightarrow \mathbb{R}, x \mapsto f(x) is called differentiable in a \in \mathbb{R} if the limit

f'(x) := Df (a) := \frac{d f}{d x} (a) := \lim_{h\rightarrow 0} \frac{ f (a+h) - f(a)}{h}


exists. One then calls f' (x) the derivative of f at a. If f is differentiable in every point of \mathbb{R}, then one says that f is differentiable.

\subsectionDifferentiability

Definition 4.7 A function f:\mathbb{R}^2 \rightarrow \mathbb{R}, (x,y)\mapsto f(x,y) is called partially differentiable in the point (a,b) \in \mathbb{R}^2 with respect to the variable x (resp.~y), if the limit

\frac{\partial f}{\partial x} (a,b) := \lim_{h\rightarrow 0} \frac{ f (a+h,b) - f(a,b)}{h}

\Big( \: \text{resp. } \frac{\partial f}{\partial y} (a,b) := \lim_{h\rightarrow 0} \frac{ f (a,b+h) - f(a,b)}{h} \:\:\: \Big)
exists. One then calls \frac{\partial f}{\partial x} (a,b) and \frac{\partial f}{\partial y} (a,b) the partial derivatives of f at (a,b). If f is partially differentiable in every point of \mathbb{R}^2 with respect to the variables x and y, then one says that f is partially differentiable. A function f:\mathbb{R}^2 \rightarrow \mathbb{R}, (x,y)\mapsto f(x,y) is called twice partially differentiable, if it is partially differentiable, and if the partial derivatives \frac{\partial f}{\partial x} and \frac{\partial f}{\partial y} are partially differentiable as well. A function f:\mathbb{R}^2 \rightarrow \mathbb{R}, (x,y)\mapsto f(x,y) is called differentiable in the point (a,b) \in \mathbb{R}^2, if there exists a linear function L:\mathbb{R}^2\rightarrow \mathbb{R} such that

\lim_{(x,y) \rightarrow (a,b)} \frac{E(x,y)}{\sqrt{(x-a)^2 + (y-b)^2}} =0,
where E:\mathbb{R}^2\rightarrow \mathbb{R} is the error function defined by E(x,y) := f(x,y)-f(a,b) -L(x,y). One then calls L the linear approximation of f at (a,b), and writes

f(x,y) \approx f(a,b) + L (x,y) .




Definition 4.8\hspace-3mm^\star A function f:\mathbb{R}^n \rightarrow \mathbb{R}, x\mapsto f(x) is called partially differentiable in the point a=(a_1,\cdots , a_n) \in \mathbb{R}^n with respect to the variable x_i, if for every i, 1 \leq i \leq n the limit

\frac{\partial f}{\partial x_i} (a) := \lim_{h\rightarrow 0} \frac{ f (a_1, \cdots, a_i+h, \cdots , a_n) - f(a_1,\cdots , a_n)}{h}
exists. One then calls \frac{\partial f}{\partial x_i} (a) the (i-th) partial derivative of f at a with respect to the variable x_i. If f is partially differentiable in every point of \mathbb{R}^n with respect to the variables x_1,\ldots,x_n, then one says that f is partially differentiable. A function f:\mathbb{R}^n \rightarrow \mathbb{R}, x\mapsto f(x) is called twice partially differentiable, if it is partially differentiable, and if the partial derivatives \frac{\partial f}{\partial x_i}, 1\leq i \leq n are partially differentiable as well. A function f:\mathbb{R}^n \rightarrow \mathbb{R}, x\mapsto f(x) is called differentiable in the point a \in \mathbb{R}^n, if there exists a linear function L:\mathbb{R}^n\rightarrow \mathbb{R} such that

\lim_{x \rightarrow a} \frac{E(x)}{\sqrt{(x_1-a_1)^2 + \ldots + (x_n-a_n)^2}} =0,
where E:\mathbb{R}^n\rightarrow \mathbb{R} is the error function defined by E(x) := f(x)-f(a) -L(x). One then calls L the linear approximation of f at a, and writes

f(x) \approx f(a) + L (x) .




Theorem 4.9 If f:\mathbb{R}^n \rightarrow \mathbb{R}, x\mapsto f(x) is differentiable at a \in \mathbb{R}^n, then f is partially differentiable and continuous at a.



Theorem 4.10 If f:\mathbb{R}^n \rightarrow \mathbb{R}, x\mapsto f(x) is twice partially differentiable, and the second partial derivatives

\frac{\partial^2f}{\partial x_i \partial x_j} := \frac{\partial }{\partial x_i} \frac{\partial f}{ \partial x_j}
are continuous, then

\frac{\partial^2f}{\partial x_i \partial x_j} = \frac{\partial^2f}{\partial x_j \partial x_i}
for 1\leq i,j\leq n.



Definition 4.11 If f:\mathbb{R}^n \rightarrow \mathbb{R}, x\mapsto f(x) is a partially differentiable function, and x\in \mathbb{R}^n, the vector

\operatorname{grad} f (x) := \left( \frac{\partial f}{ \partial x_1} (x) , \cdots , \frac{\partial f}{ \partial x_n} (x) \right)
is called the gradient of f at x. Given a function f:\mathbb{R}^n \rightarrow \mathbb{R}, x\mapsto f(x) which is partially differentiable (up to isolated points), a point x\in \mathbb{R}^n is called a critical point of f, if \operatorname{grad} f (x) =0 , or if \operatorname{grad} f (x) is not defined.
